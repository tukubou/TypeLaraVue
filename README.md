# Laravel + Vue + Typescript Example

Laravel 5.6
Vue 2.5.7
Typescript 2.7.1

## Install

1. Clone the project to local

  ```
  git clone https://github.com/121there/laravel-vue-typescript-example.git
  cd laravel-vue-typescript-example
  ```

2. Install the dependencies

  ```
  composer install
  npm install
  ```

3. Compiling Assets

  ```
  npm run dev
  ```

4. Visit the site
